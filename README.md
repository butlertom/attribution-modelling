# attribution-modelling

This repository considers a marketing dataset to explore attribution modelling
approaches.

# Data

Data was sourced from a Dropbox account holding a [Markov Chain Attribution
Dataset](https://www.dropbox.com/scl/fo/jrw7atq517jxzqrn2gxz5/ALfzBkRA90d2z7UmLcLqQRs?rlkey=6qg8wfcdrwuy9kfya6kejcq11&e=1&dl=0).
That data set contains 586,737 touch points for 240,108 unique cookie IDs,
17,639 conversions and $110,231 in conversion value.
